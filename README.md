## Project Description
This project involved transforming a basic HTML portfolio into a fully styled, responsive website using modern CSS techniques. The goal was to create a professional-looking portfolio that adapts seamlessly across mobile, tablet, and desktop devices while maintaining good design principles and usability.

---

## Technologies Used
- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Google Fonts  
- Responsive Design 

---

## Features
- Responsive navigation bar with mobile menu  
- Flexible card layout using Flexbox  
- Responsive project grid using CSS Grid  
- Typography system using CSS variables  
- Consistent color system with custom properties  
- Styled buttons with hover effects  
- Form validation using `:focus` and `:invalid` states  
- Responsive image handling  
- Multi-column footer layout  
- Magazine-style grid layout  
- Interactive UI with transitions and animations

## Lessons Learned
- How to structure a scalable CSS system using variables
- The importance of the box model and box-sizing: border-box
- When to use Flexbox vs CSS Grid
- Mobile-first responsive design principles
- How to create reusable UI components (cards, buttons)
- Improving user experience with hover, focus, and transitions

## Challenges Faced
1. Responsive layout issues: Fixed by adopting a mobile-first approach and using flexible units
2. Grid vs Flexbox confusion: Resolved by understanding Flexbox for 1D layouts and Grid for 2D layouts
3. Maintaining consistent spacing: Solved using spacing variables
